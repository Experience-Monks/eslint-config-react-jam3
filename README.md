## Installation

```bash
npm install --save-dev eslint-config-react-jam3
```

## Usage

Once installed `eslint-config-jam3` add `jam3` in your `extends` section of your [ESLint configuration](https://eslint.org/docs/user-guide/configuring)

```
{
  "extends": ["...","react-jam3"],
  "rules": {
    // ...
  }
}
```
